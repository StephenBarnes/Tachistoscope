This is a [tachistoscope](https://en.wikipedia.org/wiki/Tachistoscope)-like tool for memorizing ordered sequences like the alphabet or months of the year. It can also be used for learning matching pairs (like that July is the 7th month and R is the 18th letter of the English alphabet) with fast recall. Tachistoscopes are especially useful if you want fast, fluent recall of long ordered sequences of short items. For other use cases, consider using a spaced repetition system instead.

Coded mostly by GPT-5.4 Codex.
